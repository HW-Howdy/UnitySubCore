# Unity_SubCore

useful development kit for unity engine

You must import the following package for use
---
>com.unity.nuget.newtonsoft-json

(Package Manager -> Add package by name)

Currently offers 7 modules.
---
 *  BT (Behavior Tree)
 *  Easing
 *  EventBus
 *  Json
 *  Pool
 *  Resolve
 *  Singleton
